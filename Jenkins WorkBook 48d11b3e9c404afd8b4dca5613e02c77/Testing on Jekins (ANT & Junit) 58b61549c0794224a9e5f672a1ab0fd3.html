Testing on Jekins (ANT & Junit)

Dashboard → Manage Jenkins → Global Tool Configuration → Ant

Make sure you have Ant installed on your Jenkins Server

which ant

if not, go and install it (CentOS)

sudo yum install ant ant-junit

Then go to ls /usr/share/ant and verify if the ant lib folder is located there and if yes put that path on the ant configuration screen and click save

Then go and create a new item (free style project)

On configuration → Source Code Management: Enable Git and past the repo url

On Build Steps → Add build step and select Invoke Ant

Select the Ant version, previously configured and as target place junit

Go to Post-build Actions and Add build step, there place the path where you’re tests are saved and also make sure this matches the path on the build.xml file for ant

Click save and run the job

After running the job a few times you can see the History
